# DeMic 命令行使用说明

## 概述

DeMic 是一个麦克风控制工具，支持通过命令行参数控制麦克风的开关状态。程序采用单实例运行模式，当已有实例在运行时，新的命令行调用会将指令发送给正在运行的实例。

## 命令行语法

```cmd
DeMic.exe [/option | -option]
```

## 可用选项

### 1. `/silent` 或 `-silent`

静默模式启动，不显示系统托盘通知。

**用途：** 启动程序但不显示"程序正在系统托盘运行"的通知气泡。

**示例：**

```cmd
DeMic.exe /silent
DeMic.exe -silent
```

**行为：**

- 首次运行：静默启动程序，不显示通知
- 已有实例运行：无操作

---

### 2. `/on` 或 `-on`

打开麦克风（取消静音）。

**用途：** 将麦克风设置为非静音状态。

**示例：**

```cmd
DeMic.exe /on
DeMic.exe -on
```

**行为：**

- 首次运行：启动程序并打开麦克风，不显示通知
- 已有实例运行：发送打开麦克风指令给运行中的实例
- 如果麦克风已经是打开状态，则不执行任何操作
- 执行成功后会播放"麦克风开启"提示音（如果已配置）

---

### 3. `/off` 或 `-off`

关闭麦克风（静音）。

**用途：** 将麦克风设置为静音状态。

**示例：**

```cmd
DeMic.exe /off
DeMic.exe -off
```

**行为：**

- 首次运行：启动程序并关闭麦克风，不显示通知
- 已有实例运行：发送关闭麦克风指令给运行中的实例
- 如果麦克风已经是关闭状态，则不执行任何操作
- 执行成功后会播放"麦克风关闭"提示音（如果已配置）

---

### 4. `/toggle` 或 `-toggle`

切换麦克风状态（开/关）。

**用途：** 在静音和非静音之间切换麦克风状态。

**示例：**

```cmd
DeMic.exe /toggle
DeMic.exe -toggle
```

**行为：**

- 首次运行：启动程序并切换麦克风状态，不显示通知
- 已有实例运行：发送切换指令给运行中的实例
- 如果当前是静音状态，切换为非静音并播放"开启"提示音
- 如果当前是非静音状态，切换为静音并播放"关闭"提示音

---

## 特殊说明

### 单实例模式

DeMic 确保同一时间只有一个实例在运行。当用户尝试启动第二个实例时，新实例会将命令发送到已运行的实例，然后自动退出。


### 配置文件支持

程序支持通过配置文件 `DeMic.ini` 预设命令行参数：

#### `[CmdLineArgs]` 节

- **CmdLineArgs**：首次启动时使用的默认命令行参数
- **CmdLineArgs2**：已有实例运行时使用的命令行参数

当不带参数启动时，程序会根据当前是否已有实例运行来决定使用哪个命令行参数执行相应的命令。

当启动时指定了命令行参数，配置文件中的命令行设置将被忽略。

**示例配置：**

```ini
[CmdLineArgs]
CmdLineArgs=/silent
CmdLineArgs2=/toggle
```

**行为：**

- 当不带参数启动 `DeMic.exe` 时
  - 如果没有实例运行：使用 `CmdLineArgs` 的值（如 `/silent`）
  - 如果已有实例运行：使用 `CmdLineArgs2` 的值（如 `/toggle`）

> [!NOTE]
> 配置文件 `DeMic.ini` 与可执行文件位于同一目录下。 详见[配置文件使用指南](config.md)。

### 自动静默标志

所有麦克风控制命令（`/on`, `/off`, `/toggle`）都会自动应用静默模式标志，不会显示系统托盘通知。

## 使用场景示例

### 1. 开机自动启动（静默模式）

在注册表中配置：

```text
键：HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
值名：DeMic
值数据："C:\Path\To\DeMic.exe" /silent
```

### 2. 创建快捷方式快速切换麦克风

创建桌面快捷方式，目标设置为：

```cmd
"C:\Path\To\DeMic.exe" /toggle
```

### 3. 通过脚本控制麦克风

批处理文件示例：

```cmd
@echo off
REM 关闭麦克风
"C:\Path\To\DeMic.exe" /off

REM 等待5秒
timeout /t 5

REM 打开麦克风
"C:\Path\To\DeMic.exe" /on
```

### 4. 配合其他软件使用

通过配置文件设置双击行为：

- 首次双击启动程序
- 再次双击切换麦克风状态

## 注意事项

1. 命令行参数前缀可以为`/`或 `-`，二者完全等价
2. 参数本身区分大小写（都是小写）
3. 每次只能指定一个参数，不支持组合参数
4. 无参数启动时会根据配置文件中的设置执行相应的命令
